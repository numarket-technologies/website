## What We Do

Most companies already have the data that could transform their decisions. It sits in spreadsheets, project management systems, HR tools, and financial software — disconnected, stale, and invisible to the people who need it most.

numarket connects to those data sources, cleans and structures the data, builds live dashboards that update automatically, and delivers ongoing analysis that explains what the numbers mean and what to do about it.

We are not a staffing firm. We are not a technology vendor. We are a data intelligence partner. The product is clarity.

1. **Discovery** — We audit your existing data sources, understand your key decisions, and map what data you have to what leadership actually needs to see.

2. **Connect & Clean** — We connect to your systems — spreadsheets, databases, ERPs, project platforms — and build clean, reliable data pipelines that refresh automatically.

## Deployment

To deploy to Vercel:

1. Create a new GitHub repository
2. Add `index.html` to the repository root
3. Connect the repository to Vercel
4. Deploy — no build configuration needed

For a custom domain, add it in the Vercel dashboard under Project Settings → Domains.

To wire up the contact form to send real emails, the recommended approach is [Formspree](https://formspree.io) or [Netlify Forms](https://www.netlify.com/products/forms/) — both work with static HTML and require only a minor update to the form's `action` attribute.
